= Goldberg 0.0.1

A CI server. With pipelines!

== Setting up a new repository

   bin/goldberg add <git url> <name>


== Running Goldberg

   bin/goldberg start [port=3000]                      # Starts the CI server and the web front-end at localhost:3000, specify port to override


Sample Goldberg Server: http://goldberg.c42.in
